#
# Safe to commit: regex-only (no literal key values). Operator must rotate keys at
# https://developer.adzuna.com/ BEFORE running filter-repo.
#
# Usage: git filter-repo --force --replace-text config/filter-repo-adzuna-replacements.txt
#
# Add literal lines in a local-only file if TruffleHog still reports verified hits
# after regex purge (never commit real key values).
