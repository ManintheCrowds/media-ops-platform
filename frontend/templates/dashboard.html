About This Platform

This self-hosted platform provides unified access to multiple services. All services are integrated with single sign-on (SSO) authentication.

Service Status Indicators

  • Online - Service is running and responding
  • Offline - Service is not responding or has errors
  • Not Registered - Service is not registered in the system

Using Services

Click the "Open" button on any service card to access that service in a new tab. You'll be automatically authenticated if SSO is configured.

System Metrics

The System Information section shows:

  • Services: Total number of registered services
  • Healthy: Number of services currently online
  • Unhealthy: Number of services with issues
